What Is The Best Programming Language for YOU in 2019 (w/ Full Stack Dev “Stonic”)

(Stonic): What’s the best programming language for you to start with? Short answer: none just pick one. Long answer…

(Shadow): What type of person you are? Or what’s your interest?

(Stonic): Will you like to this as a career or a hobby? If so you really need to ask yourself this question. Rather you're lazy, productive or somewhere in the middle. This will really help you with picking a language to start with.

(Stonic): What programming is right for you?

(Shadow): For my career people, honestly try picking a language that dedicated far deeply towards your career. For an example, if you want to have a career towards game development, Programming and designing your own game I would point you to C++, C#, and/or Java. These are amazing languages used for all sorts of games these days, but if you’re starting small and these languages scare you not to worry Game engines are a thing, they use their own API to interact with these languages to make development easier on you. Or you can use Engines that use their own languages like Godot and GameMakerStudio which uses GDScript and GML.

(Shadow): For my hobby people, try picking a language that surrounds yourself with your project. For an example, if you want to make a blog or a website to show off your creations I would point you to Javascript and PHP these languages are most commonly used in Web Development these days. But there’s also back-end or server languages like, Node, Java, Python, and Go that will also help you build out your website or blog interacting with the API and Database of the application.

(Stonic): Remember, these are only just examples you can do whatever. At the end of the day, you just picked one that you suits you best or puts money in your wallet.
