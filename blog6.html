JavaScript is one language that is on the rise. If you read my previous blog about Front-end Development trends for 2019, then you would know why. Courses/YouTube

“Project-based learning” is one of the best ways to learn JavaScript. Once you build something in JavaScript, you learn a lot about how it works and what syntax it uses. Here are some project based learning courses.

  • Traversy: Uses Project based learning and multimedia strategies to teach
  • FreeCodeCamp: Offers 300 hours of JavaScript curriculum and has a built in IDE
  • Sololearn: Offers a quick way to learn JS. Has a less in depth but just as good curriculum. Good for learning the basics.

One of the easiest ways to remember how to code in JavaScript, it to take notes on paper. Take a simple spiral notebook or journal and write in it what you learned for day, everyday of the month. At the end of the month, you will have built your own little reference guide that will help you in the future.
